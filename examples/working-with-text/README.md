Text manipulation scenarios aligned with the "Working with Text" docs. All
scripts operate on `examples/Showcase.pdf` and emit results to
`output/working-with-text/`.

- `01_find_and_replace.py` – replace the first paragraph with a new message.
- `02_redact_phrases.py` – delete paragraphs containing forbidden phrases.
- `03_highlight_matches.py` – recolor matches for a regex pattern.
- `04_move_text.py` – reposition a paragraph using absolute coordinates.
- `05_change_font.py` – restyle the document title.
- `06_add_watermark.py` – apply a DRAFT watermark across pages.

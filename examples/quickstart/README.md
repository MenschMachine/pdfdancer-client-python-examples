Quickstart examples for Showcase.pdf. Each script has a `run_example()` helper so
you can import snippets elsewhere or run them directly.

- `01_inspect_document.py` – inspect pages, paragraphs, images, and form fields.
- `02_extract_text.py` – dump every paragraph into `output/quickstart/extracted_text.txt`.
- `03_duplicate_page.py` – clone page 0 and save an updated PDF.
